# **App Name**: Systemic Identity

## Core Features:

- Dual-State Environment: A high-performance theme engine allowing users to toggle between a 'System Dark' and 'System Light' mode seamlessly.
- Schematic Profile: Expressive hero section containing developer name, professional portrait container, and a high-legibility bio section.
- Badge Grid Tech Stack: Spacious, non-cramped display of technical skills using modern, color-coded badges for backend technologies.
- Project & Contribution Vault: Bento-style gallery for projects and open-source contributions with direct GitHub deep-linking and clean margins.
- Impact Synthesizer: An AI tool that analyzes specific pull requests or repositories to generate concise, high-impact bullet points for technical summaries.
- MDX-Powered Journal: A focused blog engine integrated into the main navigation for publishing technical articles with clear typographical hierarchy.
- CI/CD Deployment Pipeline: Pre-configured GitHub Actions workflow for automated build and deployment to GitHub Pages.
- CV Access Terminal: A focused, one-click access point for viewing and downloading the latest resume in a clean PDF format.

## Style Guidelines:

- Adaptive color palette: Dark Mode uses Night Blue (#0D1117) with Indigo accents; Light Mode uses a clean Paper White (#F8F9FA) with deep Graphite (#24292F) for text.
- Terminal-inspired yet highly legible font pairing: 'JetBrains Mono' for headers and UI elements to provide a developer feel, and 'Inter' for body text to ensure maximum readability.
- Breathable bento-grid layout with generous whitespace and 32px padding between sections to prevent visual cramping and ensure a modern, airy feel.
- Minimalist, geometric SVG icons with a consistent line weight that adapt their stroke color based on the active theme.
- Smooth 300ms transitions for theme switching and subtle hover lifts on project cards to reinforce the interactive nature of the site.
